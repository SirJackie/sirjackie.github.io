Python 读写 Excel 文件

  • xlrd库：读取excel文件，支持xls和xlsx格式（最新版好像不支持xlsx，解决方法可自行搜索）。
  • xlwt库：写入excel文件，只支持xls格式。
  • openpyxl： openpyxl是用于读取/写入Excel 2010 xlsx/xlsm文件的Python库，也就是说openpyxl这个Python库不支持xls文件的读取和操作，如果在工作中遇到xls文件我们就不能使用这个库。
  • pandas（Python Data Analysis Library）：支持读写，支持xls、xlsx（依赖于openpyxl或者xlrd，会自动判断文件格式，并选择相应的库）。
