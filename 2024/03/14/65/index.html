吐槽一下 Android 开发要下载的东西之多

盘点一下要创建一个安卓空项目需要下载的东西：

Android Downloads

  • JDK 8 (200MB)
  • Android Studio (2GB)
  • Android SDK Platforms (15GB)
  • Android SDK Tools (10GB)
  • Android AVD (~5GB)
  • Gradle etc. when creating the first project (~1GB)

差不多33.2GB，而且普通网络还下不了，需要用魔法才能连接到需要的网络😅

不多说了，上几张图：

附上Gradle下载的日志，方便大家感受一下，究竟是多么恐怖：

不过，论下载依赖文件的恐怖程度，安卓开发只能排第二，排第一的当之无愧是nodejs和npm。

俗话说，nodejs的依赖比黑洞还深。你想下一个UI库，结果拖家带口1400+个依赖。有的依赖还依赖外部语言，比如Python、Java和C++，所以JDK的Python Interpreter也会被下载下来。最无语的是，为了编译C++，nodejs能把整个Visual Studio 2019作为一项依赖给下载到电脑里去，真是无语。

nodejs一般依赖可以轻松占用17-20GB😢，安卓比起来，还是小巫见大巫，too young too simple了，哈哈😃。
